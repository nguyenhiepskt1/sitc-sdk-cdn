📚 Integration Guide

To use SITCSDK in your project, simply include the script from JSDelivr:

<!-- Include SITCSDK Mock from JSDelivr --> <script src="https://cdn.jsdelivr.net/gh/nguyenhiepskt1/sitc-sdk-cdn@main/sitc-sdk-mock.js"></script> <script> // Use SITCSDK APIs const location = await window.sitcSDK.getLocation() const photo = await window.sitcSDK.takePhoto() await window.sitcSDK.sendNotification('Hello from my app!') window.sitcSDK.showSuccess('Operation completed!') // Get user information const userInfo = await window.sitcSDK.getUserInfo() console.log('User:', userInfo.user) console.log('Permissions:', userInfo.permissions) </script>

  • ✅ Notifications: Send local notifications with scheduling
